Work to do

Part 1:

The classical Machine Learning Part

    Exercise 1:

      Cut each file of each language into two parts containing an equal number of lines (for example, we cut the file fr.txt into a file train.fr and a file test.fr )

      ·    Q1: Why should we use different texts for learning and testing?

      Recall: The linux wc command gives the number of lines of a text file; the head (or tail ) command allows you to retrieve the first n (or last) lines of a file.

    Exercise 2:

      Construct a trigram language model for each training file previously created using the following commands:

      ngram-count -order 3 -text train.fr -write fr3g.bo -write-vocab fr.voc

      ngram-count -read fr3g.bo -lm fr3g.BO -gt2min 1 -gt3min 2 

      ·    Q1: Observe files .voc , .bo and .BO ; what do they contain?

      ·    Q2: In your opinion, what do the symbols <s> and </s> correspond to?

      ·    Q3: What are the -gt2min and -gt3min options for ? In what cases are they useful / necessary?

      ·    Q4: What is the most common trigram in each language?

      Note: The first column of the .BO file is homogeneous at the logarithm of a probability.

      Help: The unix sort -n command allows you to sort a file according to one of its fields present in numerical form.

    Exercise 3:

      Calculate the perplexity of each text test.LANG compared to each model LANG3g.BO . Example of using the ngram command:

      ngram -lm fr3g.BO -ppl test.en

      ·    Q1: Present the results in the form of a matrix.

      ·    Q2: How can we build a language recognition system for a text using a stochastic language model?

    Exercise 4:

      Redo the same system with n-gram CHARACTER models.

      ·    Q1: What are the perplexities for this character model? What might be the advantages / disadvantages of the character model compared to the word model?

      ·    Q2: Redo the model by replacing the space with any symbol (eg, <SPACE>). What are the perplexities for this model?

      Help: One model can be considered more efficient than another if, for a text in the same language, the perplexity is lower and, for a text in another language, the perplexity is higher.

    Exercise 5:

      Extend the perlexity matrix for all other languages.

      ·    Q1: Check which language pairs are the easiest / hard to distinguish from each other

      ·    Q2: Redo the model by replacing the space with any symbol (eg, <SPACE>). What are the perplexities for this model?
